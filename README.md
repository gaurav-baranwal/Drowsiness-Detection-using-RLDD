# Drowsiness-Detection-using-RLDD

Yawning and Drowsiness Detection with alarm alert using DLIB 

Simple code in python to detect Drowsiness and Yawn and alert the user using Dlib.

## Dependencies

1. Python 3.8.3
2. opencv (v 4.4.0.46) 
3. dlib	(v 19.21.1)
4. imutils (v 0.5.3)
5. scipy (v 1.6.0)
6. numpy (v 1.19.3)
7. argparse
